# Personal Portfolio

A clean, minimal portfolio website built to showcase my work, skills, and approach to development.

---

## ✦ 🎨 Design

The design focuses on simplicity, clarity, and strong visual structure.

- 🧼 **Minimalist layout** — no unnecessary elements, content-first approach  
- 🌑 **Dark aesthetic** — modern, low-strain visual experience  
- 🧠 **Typography-driven** — clear hierarchy and readability  
- ✨ **Subtle motion** — lightweight animations for better UX  
- 📱 **Responsive** — works across desktop and mobile  

---

## ✦ 📌 Overview

This site presents:
- 🧩 Projects and work  
- ⚙️ Technical skills  
- 📬 Contact information  

Built using plain HTML, CSS, and JavaScript for maximum performance and control.
