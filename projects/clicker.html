How It Works

This is a simple clicker game built using HTML, CSS, and JavaScript. Users can click to gain points, buy upgrades, and reset the game. The game data is saved in the browser's local storage, so it persists even after a page reload.

Game Features:

  • Click the button to earn points.
  • Buy upgrades to increase your points per click or automatically earn points.
  • Save your progress and resume later using your browser's local storage.
