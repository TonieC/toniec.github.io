Featured Project

tflows.py

A lightweight Python framework for building Discord bots and task automation pipelines. Designed around a scripting engine that keeps bot logic modular, testable, and separate from Discord's event system.

Python AsyncIO Discord API Open Source PyPI
GitHub PyPI Documentation
Architecture Overview
  • Async-first design built on Python's asyncio — event handlers and tasks run concurrently without blocking
  • Scripting engine separates bot commands from Discord plumbing, making individual flows testable in isolation
  • Modular loader system lets you add, remove, or reload bot modules at runtime without restarting
  • Thin abstraction layer over discord.py — doesn't hide the library, just reduces repetitive wiring
Real Use Cases
Server Moderation Event-driven auto-moderation rules without custom bot architecture
Task Automation Scheduled jobs and webhook triggers wired to Discord channels
Rapid Prototyping Stand up a functional bot in minutes with minimal boilerplate
Scripting Engine Define flows as plain functions; the framework handles wiring
